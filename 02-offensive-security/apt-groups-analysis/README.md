# Analysis of Major Threat-Actor Groups

> Part of the **GoIT Cybersecurity Specialist** program (2024–2025).

## Objective
Study the tooling, objectives, and methodologies of three well-known threat groups: Anonymous, APT29 (Cozy Bear), and Equation Group.

## Tools & Technologies
Threat intelligence, MITRE ATT&CK concepts

## Skills Demonstrated
Threat actor analysis, TTP mapping

## Outcome
A comparative profile of three groups with recommended defensive measures.
