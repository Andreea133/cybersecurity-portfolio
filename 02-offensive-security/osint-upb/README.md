# OSINT Reconnaissance of a University Network

> Part of the **GoIT Cybersecurity Specialist** program (2024–2025).

## Objective
Map the public IT infrastructure of a university (UPB) using open-source intelligence: public servers, operating systems, open ports, services, and known CVEs.

## Tools & Technologies
Shodan, Nmap, Sublist3r, whois, dig, CVE Details

## Skills Demonstrated
OSINT, reconnaissance, vulnerability research

## Outcome
An infrastructure map with identified services and associated known vulnerabilities.
