# Firewall Configuration with pfSense

> Part of the **GoIT Cybersecurity Specialist** program (2024–2025).

## Objective
Build a virtual network and configure pfSense as a firewall to block access to specific domains (e.g. Facebook) using traffic rules.

## Tools & Technologies
pfSense, VirtualBox, firewall rules, VLANs

## Skills Demonstrated
Network security, firewall administration, virtualization

## Outcome
A working firewall lab that successfully blocks targeted domains, verified end-to-end.
