> Part of the **GoIT Cybersecurity Specialist** program (2024–2025).

Design a Security Operations Center (SOC) for a Tesla-scale company, defining security needs across connected vehicles, manufacturing, and user data.

## Tools & Technologies
SOC frameworks, SIEM concepts, threat modeling

## Skills Demonstrated

## Outcome
A SOC blueprint covering monitoring priorities, alerting, and response for a large multi-sector target.
