# Multi-Factor Authentication System (Thesis Contribution)

> Part of the **GoIT Cybersecurity Specialist** program (2024–2025).

## Objective
Design and implement an email-based One-Time-Password (OTP) MFA system as a personal contribution within the bachelor's thesis, and compare it with biometric authentication.

## Tools & Technologies
Python, email OTP, MFA, cryptography

## Skills Demonstrated
Authentication design, secure development, applied crypto

## Outcome
A working email-OTP MFA prototype with time-limited, single-use codes, analyzed against biometric approaches.
